# GreenKey

Небольшая настольная программа (Windows, Python + Tkinter) для удаления **зелёного фона** с картинок/принтов — аналог связки **Keylight + Advanced Spill Suppressor** из After Effects.

## Возможности
- Кей зелёного фона по алгоритму Keylight (Screen Colour RGB 11,163,77, Clip Black 9 / Clip White 76).
- Подавление зелёного перелива (Advanced Spill Suppressor, Standard, Suppression 100).
- Гибридная обработка края: тёмное → navy, светлое → чистый цвет (деконтаминация).
- Антиалиасинг с рендером в повышенном разрешении (до 8K при сохранении).
- Просмотр: зум колесом мыши, панорама зажатым колесом.
- Пакетная обработка папки, экспорт PNG с прозрачным фоном.

## Запуск
Двойной клик по `GreenKey.bat` (нужен установленный Python с пакетами `numpy` и `Pillow`).

## Файлы
- `GreenKey.py` — программа
- `GreenKey.bat` — запуск без окна консоли
- `ЧИТАЙ_МЕНЯ.txt` — инструкция
