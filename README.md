# GreenKey

Небольшая настольная программа (Windows, Python + Tkinter) для удаления **зелёного или синего фона** с картинок/принтов — аналог связки **Keylight + Advanced Spill Suppressor** из After Effects.

## Возможности
- **Авто-определение типа фона** (зелёный/синий) по краям кадра и кей по доминантному каналу
  (зелёный `G>max(R,B)` или синий `B>max(R,G)`), не трогая жёлтый/бирюзу/телесный.
- Кей по алгоритму Keylight: зелёный Screen Colour RGB 11,163,77 или синий RGB 0,89,255,
  Clip Black 9 / Clip White 76 (параметры из пресетов `Green.ffx` / `blue.ffx`).
- Подавление перелива (Advanced Spill Suppressor, Standard, Suppression 100).
- Гибридная обработка края: тёмное → navy, светлое → чистый цвет (деконтаминация).
- Антиалиасинг с рендером в повышенном разрешении (до 8K при сохранении).
- Просмотр: зум колесом мыши, панорама зажатым колесом.
- **Пакетная обработка папки** в фоновом потоке (окно не виснет) с прогресс-баром,
  вывод в новую датированную папку `ГГГГ-ММ-ДД_01`.
- **Яндекс.Диск через вход в аккаунт (OAuth)**: войти своим аккаунтом, указать
  входную папку (публичной ссылкой или путём) и выходную папку на Диске — программа
  скачает, уберёт фон, зальёт результат в датированную подпапку и вернёт публичную ссылку.
- **Облако (Яндекс.Диск / Облако Mail.ru) через rclone**: скачать папку из облака,
  убрать фон, залить результат обратно и получить публичную ссылку.

## Запуск
Двойной клик по `GreenKey.bat` (нужен установленный Python с пакетами `numpy` и `Pillow`).

## Янд.Диск через вход в аккаунт (без rclone)
Один раз зарегистрируйте приложение на [oauth.yandex.ru/client/new](https://oauth.yandex.ru/client/new):
платформа «Веб-сервисы», Redirect URI `http://localhost:8123`, доступы к Яндекс.Диск REST API
(чтение/запись/инфо) и к логину. Вставьте ClientID и Client secret в панель «Яндекс.Диск»,
нажмите «Войти в Яндекс», укажите входную папку (ссылка или `/путь`) и выходную (`/путь`),
затем «Я.Диск: вход → выход». Токен хранится локально в `yandex_config.json` (в репозиторий не попадает).

## Облако через rclone (Яндекс + Mail.ru)
Для работы с облаком установите [rclone](https://rclone.org/downloads) и настройте аккаунты:

```
rclone config          # создать remote типа "yandex" и/или "mailru"
rclone listremotes     # проверить: yandex:  mailru:
```

Затем в программе укажите папку-источник вида `yandex:Принты` или `mailru:foto`
и нажмите «Облако → вырезать всё». Аутентификацию делает сам rclone — программа паролей не хранит.
